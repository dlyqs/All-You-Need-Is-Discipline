# 用户基础背景信息

这份文件只记录长期稳定的用户背景和交易纪律。市场主线、热点题材、每日行情和临时观点不要写到这里。

## 基础信息

| 字段 | 值 | 备注 |
| --- | --- | --- |
| 主要交易市场 | 待填写 | 例如 A股、美股 |
| 账户币种 | 待填写 | 例如 CNY、USD |
| 资金规模或记录方式 | 待填写 | 可以写总资金、可用资金口径 |
| 风险偏好 | 待填写 | 例如 保守、均衡、进攻 |
| 最低现金保留规则 | 至少保留 30% 现金 | 默认纪律，可直接修改 |

## 交易纪律备注

- 不因卖飞或热点刺激开盘乱追。
- 不同一题材板块买多只，优先保留股性最好的一只。
- 不把模型输出当成自动下单指令，最终交易由用户确认。
